# Lambda + Boto3 Python Automation

This repository contains Python scripts using Boto3 to automate AWS tasks.  
You can create VPCs, subnets, IAM users, EC2 instances, EBS volumes, and manage S3 buckets.  

## Scripts included
- aws_vpc_setup.py
- aws_iam_setup.py
- start_ec2.py / stop_ec2.py / terminate_ec2.py
- create_volume.py / describe_volume.py
- s3_bucket.py
